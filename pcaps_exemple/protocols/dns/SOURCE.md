# Provenance

Toutes les captures de ce dossier proviennent du dépôt public de
**Chris Sanders** : <https://github.com/chrissanders/packets>
(contact : chris@chrissanders.org — blog : <http://www.chrissanders.org>).

| Fichier | Contenu |
|---|---|
| `dns_query_response.pcapng` | requête/réponse DNS simple (port 53) |

| `dns_recursivequery_client.pcapng` | résolution récursive, vue client |
| `dns_recursivequery_server.pcapng` | résolution récursive, vue serveur |
| `dns_reverse_lookup.pcapng` | requête PTR (reverse lookup) |
| `dns_tcp.pcapng` | DNS sur TCP |
| `dns_axfr.pcapng` | transfert de zone (AXFR) |
| `dns_lab.pcapng` | trafic DNS de lab |
| `dns.pcap` | trafic DNS générique (format pcap classique) |
| `udp_dnsrequest.pcapng` | requête DNS UDP isolée |
| `dns_isp_hijack.pcapng` | réponse DNS détournée par un FAI |
